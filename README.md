# 🧭 Personal Portfolio Website

A modern, responsive portfolio website showcasing professional experience,
skills, and projects. Designed for clarity, minimalism, and smooth navigation
across all devices.

## ✨ Purpose & Goals

- **Professional Profile:** Present a comprehensive overview of key
  achievements.
- **Visual CV:** Serve as a dynamic online presence for recruiters and
  collaborators.
- **Skill Demonstration:** Showcase front-end development capabilities and
  design consistency.
- **User Experience:** Provide a clean, fast, and fully accessible user
  interface.

## 🧩 Key Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop
  viewports.
- **Smooth Navigation:** Section-based layout with anchor links and fluent
  transitions.
- **Performance:** Lightweight assets and highly efficient CSS structure.
- **Certificates Showcase:** Structured, chronological list of trainings and
  achievements.
- **Interactive Contact:** Social links and email with custom, subtle hover
  animations.

## 🛠 Tech Stack

- **HTML5:** Semantic markup tailored for SEO and accessibility (a11y).
- **CSS3:** Flexbox and Grid layouts ensuring precision and responsiveness.
- **Bootstrap 5:** Utility classes utilized for spacing, typography, and core
  components.
- **Custom Animations:** Tailored CSS transitions and fade-in effects for an
  interactive feel.

## 📁 Project Structure

| Section          | Description                                                     |
| :--------------- | :-------------------------------------------------------------- |
| **Header**       | Navigation bar featuring responsive anchor links.               |
| **Hero Section** | Professional title, name, and a high-impact short introduction. |
| **About Me**     | Personal description highlighting professional identity.        |
| **Skills**       | Visual grid layout displaying technical proficiencies.          |
| **Certificates** | Structured timeline detailing completed courses and trainings.  |
| **Contact**      | Direct email access and external links (LinkedIn, RG).          |
| **Footer**       | Copyright notices and project development credits.              |
